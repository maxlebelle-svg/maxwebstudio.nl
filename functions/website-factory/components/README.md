# Component Rules

Componenten zijn de bouwblokken van een websiteproduct.

Een package bepaalt of een component actief is. Een branche bepaalt de inhoudelijke en visuele richting.

Voorbeelden:

```text
Starter: hero, services, cta, contact, footer
Business: hero, services, portfolio, reviews, faq, cta, contact, footer
Premium: hero, portfolio, reviews, cases, team, leadMagnet, cta, contact, footer
```

Nieuwe modules worden toegevoegd als componentmanifest en daarna per pakket aan- of uitgezet.
