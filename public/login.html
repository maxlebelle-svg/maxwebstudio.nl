Auth architectuur

Demo blijft werken terwijl productie-auth wordt voorbereid.

  • Demo-sessies blijven lokaal in deze browser.
  • Echte login gebruikt straks Supabase Auth en RLS.
  • Geen wachtwoorden, API keys of secrets worden in de frontend opgeslagen.
  • Accountaanvragen worden tijdelijk lokaal bewaard in maxwebstudioAccountRequests.
