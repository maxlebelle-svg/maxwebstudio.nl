# Blinde beoordelingen

Plaats hier uitsluitend volledig ingevulde JSON-exports uit het Gold Set-beoordelingsscherm.

Regels:

- minimaal twee onafhankelijke beoordelaars;
- iedere beoordelaar rondt alle 24 cases af;
- de beoordelaar ziet vóór export alleen Website A en Website B;
- bestandsnaam: `reviewer-<naam>.json`;
- bestanden moeten voldoen aan `../assessment.schema.json`;
- wijzig beoordelingen niet nadat de A/B-mapping is onthuld;
- een nieuwe beoordelingsronde vereist een nieuwe Gold Set-versie wanneer de cases veranderen.
