# Content Factory asset candidates v1

Volledig geïsoleerde, ongekoppelde beeldkandidaten voor handmatige beoordeling na Gold Set-certificering.

De package bevat het toegestane maximum van 60 kandidaten: zes per pilotbranche, verdeeld over twee heroes, twee services, één about/team en één detail/ambiance.

## Status

- Package: `candidate_unreviewed`
- Review: `unreviewed`
- Publication: `blocked`
- Linkage: `unlinked`

Open `review-gallery.html` lokaal voor de visuele review. `manifest.json` is de machineleesbare bron. `IMPORT_PROPOSAL_NOT_EXECUTED.md` beschrijft alleen een eventueel later proces en voert niets uit.

Geen bestand in deze map mag rechtstreeks door een library, adapter, renderer, Gold Set-case, productie- of stagingsysteem worden geïmporteerd.
