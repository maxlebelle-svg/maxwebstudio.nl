# SEO Guide

Dit document beschrijft SEO-richtlijnen voor Max Web Studio.

## Huidige Status

Sterk:

- alle live pagina's hebben een title
- alle live pagina's hebben een meta description
- alle live pagina's hebben een H1
- homepage heeft basis structured data
- interne links zijn aanwezig

Verbeterpunten:

- geen sitemap.xml gevonden
- geen robots.txt gevonden
- geen canonical tags
- beperkte Open Graph metadata
- beperkte structured data
- ondersteunende pagina's zijn vrij kort

## Nieuwe Pagina Checklist

Elke nieuwe pagina moet hebben:

- unieke title
- unieke meta description
- duidelijke H1
- logische H2/H3 structuur
- interne links naar relevante pagina's
- CTA naar aanvraag, gesprek of betaling
- alt-teksten bij afbeeldingen
- consistente header
- waar relevant structured data

## Metadata Richtlijnen

Title:

- kort en specifiek
- merknaam waar passend
- belangrijkste zoekterm vooraan

Meta description:

- 120-160 tekens als richtlijn
- concreet aanbod
- reden om te klikken
- natuurlijk Nederlands

## Structured Data

Huidige homepage gebruikt `ProfessionalService`.

Mogelijke uitbreidingen:

- LocalBusiness
- Service
- FAQPage
- BreadcrumbList
- Product of Offer voor pakketten

Structured data alleen toevoegen wanneer de content op de pagina dit ondersteunt.

## Contentstrategie

Kansen:

- website laten maken
- website laten maken voor zzp
- website laten maken voor lokale ondernemers
- hosting en onderhoud website
- website met Mollie betaalflow
- website onderhoud abonnement
- lokale dienstenpagina's per doelgroep

Geen dunne SEO-pagina's maken. Iedere pagina moet nuttig zijn voor echte bezoekers.

## Technische SEO

Aanbevolen verbeteringen:

- `robots.txt`
- `sitemap.xml`
- canonical tags
- Open Graph image
- betere image dimensions
- lazy loading
- performance optimalisatie
- duidelijke URL-structuur

## Conversie En SEO Samen

SEO-pagina's moeten altijd converteren via:

- Plan gratis gesprek
- WhatsApp Max
- Bel direct
- Bekijk pakketten
- Betaal/start project waar passend
